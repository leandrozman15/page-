Abratecnica — Preaprobación del nuevo sitio y app ERP

Entrega para validación visual y funcional antes de publicación definitiva.

Resumen ejecutivo

  • Diseño unificado (home, productos, tienda).
  • Integración ERP con cliente robusto (reintentos/timeout, fallback GET).
  • API local en hosting (PHP 5.6 + MySQL) para pedidos y contacto.
  • Checkout con notificación por email a internos.
  • Páginas legales y soporte actualizadas.

Estado técnico

  • API PHP lista: /api/orders.php, /api/contact.php
  • DB: abratecnica configurada (utf8mb4)
  • DNS .com: migración a Locaweb en curso
  • DNS .com.br: A y CNAME creados
  • SSL: se activará tras propagación

Accesos de prueba

  • Home (local): index.html
  • Tienda (local): tienda.html
  • Productos (local): productos.html
  • Salud API: /api/health.php

Nota: si abriste este archivo desde WhatsApp/Drive (especialmente en iPhone), los accesos locales pueden no abrirse. En ese caso, abre el ZIP en una carpeta y ejecuta el archivo dentro de la misma carpeta en un computador; o usa los accesos públicos cuando el dominio esté vinculado.
